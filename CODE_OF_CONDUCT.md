# Code of Conduct

## 1. Our Goal
To build a feature-rich Minecraft launcher that serves the user in a professional and friendly environment. We value clean code, efficiency and respect for everyone's time.

## 2. Expected Behavior
- Be direct but kind: If code is bad, be honest, but explain without attacks.
- Respect the guidelines: Don't get defensive if your code is too opinionated. To keep the project both usable and maintainable, we have to care about everyone.
- Assume competence: Everyone is an unpaid volunteer. This isn't a corporate environment where we expect people to have years of experience, but we trust people to do their best.

## 3. Unacceptable Behavior
- Gatekeeping: Do not mock anyone for any reason. This includes OS choice, versions of software or knowledge levels.
- Harassment: Personal attacks, insults, discriminatory language isn't allowed here and so is attempting to contact anyone off-platform without their consent.
- Spamming/Low Effort: We want to create a good program and contributions should be for the greater good, not for social gain of being a contributor.

## 4. Enforcement
In case you break the CoC, any maintainer may:
- Without warning, close a PR or Issue.
- Temporarily or permanently ban an account.
